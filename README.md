Ejercicios de PHP - DAW (2º Año) 🐘
Este repositorio contiene las prácticas y ejercicios realizados durante el segundo año del Ciclo Formativo de Grado Superior en Desarrollo de Aplicaciones Web (DAW) en el Instituto Provençana.

📂 Contenido del Repositorio
El proyecto está organizado por bloques temáticos según el progreso del curso:

1. Fundamentos y Lógica
Estructuras de Control: Ejercicios de condicionales (dados, IFs) y bucles.

Arrays: Manipulación y funciones avanzadas de arreglos en PHP.

Refuerzo: Ejercicios de lógica para afianzar la sintaxis básica.

2. Formularios y Gestión de Datos
Manejo de Formularios: Procesamiento de datos enviados por GET y POST.

Autenticación y Sesiones: Sistema de login, manejo de cookies y control de sesiones (login.php, logout.php, autenticación.php).

3. Programación Orientada a Objetos (POO)
Herencia: Ejemplos prácticos de extensión de clases y reutilización de código.

Modelos de Datos: Estructuras de objetos para representar entidades.

4. Arquitectura y Patrones
MVC (Modelo-Vista-Controlador): Implementación de una arquitectura profesional para separar la lógica de negocio de la interfaz de usuario (mvc1_alumnes).

Generador de Sitios Estáticos: Proyecto para la generación automatizada de contenido.

🛠️ Tecnologías Utilizadas
Lenguaje: PHP 8.x

Servidor: Apache (XAMPP / Laragon)

Arquitectura: MVC (Modelo-Vista-Controlador)

Frontend: HTML5, CSS3 (vía formularios y vistas PHP)
